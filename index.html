Importing Pandas

Pandas is a powerful data manipulation library for Python. To use it, you first need to import it into your script or notebook.

# Importing pandas library import pandas as pd

By convention, Pandas is imported with the alias pd. This makes it easier to reference Pandas functions throughout your code.

Tip: Make sure you have Pandas installed. If not, install it using pip: pip install pandas

Storing Data as CSV

After processing your data, you might want to save it back to a CSV file. Pandas makes this easy with the to_csv method.

Syntax

# Basic syntax for saving to CSV dataframe.to_csv("output.csv", index=False)

The index=False parameter prevents Pandas from writing row indices to the file.

Example

# Creating a dictionary data = { "Name": ['Ram', 'Shyam', 'Lakshman'], "Class": ['B.tech', 'M.tech', 'P.hd'], "City": ['heart', 'xyz', 'wer'] } # Creating DataFrame df = pd.DataFrame(data) # Display the DataFrame print(df) # Saving to CSV df.to_csv("output.csv", index=False)
Output:
Name Class City
0 Ram B.tech heart
1 Shyam M.tech xyz
2 Lakshman P.hd wer
Tip: After running this code, you will find the output.csv file in the same folder where your Python script is located.

Storing Data as JSON

You can also save your DataFrame as a JSON file using the to_json method. JSON is particularly useful for web applications.

Syntax

# Basic syntax for saving to JSON dataframe.to_json("output.json", index=False)

The index=False parameter prevents Pandas from including row indices in the JSON output.

Example

# Creating a dictionary data1 = { "Name": ['Ram', 'Shyam', 'Lakshman'], "Class": ['B.tech', 'M.tech', 'P.hd'], "City": ['heart', 'xyz', 'wer'] } # Creating DataFrame df1 = pd.DataFrame(data1) # Display the DataFrame print(df1) # Saving to JSON df1.to_json("output.json", index=False)
Output:
Name Class City
0 Ram B.tech heart
1 Shyam M.tech xyz
2 Lakshman P.hd wer
Tip: After running this code, you will find the output.json file in the same folder where your Python script is located.

JSON Orientations

When converting DataFrames to JSON, you can choose different orientations depending on your use case:

1. Array of Objects (orient='records')

Each object in the array represents a complete row of data.

# Create DataFrame data = { "Name": ['Ram', 'Shyam', 'Lakshman'], "Class": ['B.tech', 'M.tech', 'P.hd'], "City": ['heart', 'xyz', 'wer'] } # Convert to DataFrame df = pd.DataFrame(data) # Generate Array of Objects JSON array_of_objects = df.to_json(orient='records', indent=2) # Display the result print("Array of Objects:") print(array_of_objects)
Output:
Array of Objects:
[
  {
    "Name": "Ram",
    "Class": "B.tech",
    "City": "heart"
  },
  {
    "Name": "Shyam",
    "Class": "M.tech",
    "City": "xyz"
  },
  {
    "Name": "Lakshman",
    "Class": "P.hd",
    "City": "wer"
  }
]
Key Points:
  • Human-friendly: Easy to read (each row is a self-contained record)
  • API-friendly: Perfect for sending data to web APIs

2. Column Oriented (orient='columns' - default)

Groups data by columns (like a dictionary of columns).

# Generate Column Oriented JSON column_oriented = df.to_json(orient='columns', indent=2) # Display the result print("\nColumn Oriented:") print(column_oriented)
Output:
Column Oriented:
{
  "Name": {
    "0": "Ram",
    "1": "Shyam",
    "2": "Lakshman"
  },
  "Class": {
    "0": "B.tech",
    "1": "M.tech",
    "2": "P.hd"
  },
  "City": {
    "0": "heart",
    "1": "xyz",
    "2": "wer"
  }
}
Key Points:
  • Column-focused: Easy to access all values in a specific column
  • Preserves indices: Row numbers are kept as keys

3. Value Oriented (orient='values')

A simple array of arrays (like a spreadsheet). No column names or row indices.

# Generate Value Oriented JSON value_oriented = df.to_json(orient='values', indent=2) # Display the result print("Value Oriented:") print(value_oriented)
Output:
Value Oriented:
[
  ["Ram", "B.tech", "heart"],
  ["Shyam", "M.tech", "xyz"],
  ["Lakshman", "P.hd", "wer"]
]
Key Points:
  • Compact: Smallest file size (no metadata like column names)
  • Machine-friendly: Ideal for numerical processing

Saving Different Formats

# Save each format to separate files df.to_json("array_of_objects.json", orient='records', indent=2) # Save column oriented format df.to_json("column_oriented.json", orient='columns', indent=2) # Save value oriented format df.to_json("value_oriented.json", orient='values', indent=2)

Why Explore Data?

Data exploration is a crucial first step in any data analysis or machine learning project. It helps you:

1. Understand the Dataset

Get familiar with the structure of your data:

  • Number of rows and columns
  • Types of data
  • Names of columns

2. Identify Problems

Detect potential issues in your data:

  • Missing values
  • Incorrect data
  • Duplicate entries

3. Plan Next Steps

Based on your findings, decide on the next steps:

  • Cleaning the data
  • Transforming features
  • Selecting appropriate analysis techniques
Best Practice: Always explore your data before diving into complex analysis or modeling. This helps you avoid surprises and ensures your analysis is based on a solid understanding of the data.

Filtering Rows

Filtering rows based on conditions is a common operation in data analysis. Pandas provides several ways to do this.

Single Condition

To filter rows based on a single condition, use a boolean expression inside square brackets.

# Filter rows where age > 23 filtered_rows = df[df["age"] > 23] # Display the result print(filtered_rows)

Multiple Conditions (AND)

To filter rows based on multiple conditions with AND logic, use the & operator.

# Filter rows where age > 23 AND Performance Score > 80 filtered_rows = df[(df["age"] > 23) & (df["Performance Score"] > 80)] # Display the result print(filtered_rows)

Multiple Conditions (OR)

To filter rows based on multiple conditions with OR logic, use the | operator.

# Filter rows where age > 23 OR Performance Score > 80 filtered_rows = df[(df["age"] > 23) | (df["Performance Score"] > 80)] # Display the result print(filtered_rows)

Example

# Importing pandas import pandas as pd # Creating a dictionary data = { "name": ['Ram', 'Shyam', 'Lakshman', 'lokesh', 'manish', 'ramu'], "age": [23, 25, 21, 22, 24, 26], "Salary": ['23000', '40000', '89000', 60000, 45000, 67000], "city": ['heart', 'xyz', 'wer', 'abc', 'def', 'ghi'], "Performance Score": [85, 90, 78, 88, 76, 95] } # Creating DataFrame df = pd.DataFrame(data) # Display original DataFrame print('Original Dataframe:\n') print(df) # Filter rows based on condition print("\n") print(df[df["age"] > 23]) # Select rows based on multiple conditions (AND) print("\n") print(df[(df["age"] > 23) & (df["Performance Score"] > 80)]) # Select rows based on multiple conditions (OR) print("\n") print(df[(df["age"] > 23) | (df["Performance Score"] > 80)])

What is a Pandas Series?

A Pandas Series is a one-dimensional labeled array capable of holding data of any type (integer, string, float, Python objects, etc.).

Key Characteristics

  • One-dimensional
  • Heterogeneous data (can hold different types of data)
  • Size-immutable (after creation, size cannot be changed)
  • Values can be modified

Creating a Series

You can create a Series from various data structures:

  • List
  • Tuple
  • Dictionary
  • NumPy array
  • Scalar value
  • Range

Updating Series Values

There are several ways to update values in a Pandas Series:

1. Updating by Index Label

Use the index label to modify a specific value.

# Create a sample Series data = pd.Series([10, 20, 30, 40], index=['a', 'b', 'c', 'd']) # Display original Series print("Original Series:") print(data) # Update value at index 'c' to 99 data['c'] = 99 # Display updated Series print("Updated Series:") print(data)

b 20
c 30
d 40
dtype: int64

Updated Series:

b 20
c 99
d 40
dtype: int64

2. Updating by Position (iloc)

Use .iloc[] to update values by position (0-based index).

# Create a sample Series data = pd.Series([10, 20, 30, 40], index=['a', 'b', 'c', 'd']) # Display original Series print("Original Series:") print(data) # Update value at position 1 to 101 data.iloc[1] = 101 # Display updated Series print("Updated Series:") print(data)

d 40
dtype: int64

Use boolean indexing to update values that meet a condition.

# Create a sample Series data = pd.Series([10, 20, 30, 40], index=['a', 'b', 'c', 'd']) # Display original Series print("Original Series:") print(data) # Update values greater than 20 to 999 data[data > 20] = 999 # Display updated Series print("Updated Series:") print(data)

Use .update() to modify values in-place using another Series (aligns by index).

# Create a sample Series data = pd.Series([10, 20, 30, 40], index=['a', 'b', 'c', 'd']) # Display original Series print("Original Series:") print(data) # Create another Series for updates updates = pd.Series([111, 222], index=['b', 'd']) # Display updates Series print("Updates Series:") print(updates) # Update values using another Series data.update(updates) # Display updated Series print("Updated Series:") print(data)

5. Replacing Specific Values

Use .replace() to substitute exact values.

# Create a sample Series data = pd.Series([10, 20, 30, 40], index=['a', 'b', 'c', 'd']) # Display original Series print("Original Series:") print(data) # Replace specific values: 10 to 1000 and 30 to 3000 data.replace({10: 1000, 30: 3000}, inplace=True) # Display updated Series print("Updated Series:") print(data)

Apply a function to update values (e.g., multiply all values by 2).

# Create a sample Series data = pd.Series([10, 20, 30, 40], index=['a', 'b', 'c', 'd']) # Display original Series print("Original Series:") print(data) # Multiply all values by 2 data = data * 2 # Display updated Series print("Updated Series:") print(data)

Dictionaries in Pandas

In Pandas, dictionaries are often used to create DataFrames. A dictionary is like a translator between your raw data and Pandas, organizing data with labels and values.

Why Use Dictionaries in Machine Learning?

  • Loading data into Pandas
  • Preparing data for models
  • Saving model results

Creating a DataFrame from a Dictionary

# Importing pandas import pandas as pd # Simple dictionary with our data data = { 'size': [2500, 3000, 1500, 3500], 'bedrooms': [3, 4, 2, 5], 'price': [500000, 600000, 350000, 750000] } # Turn it into a DataFrame (like a table) df = pd.DataFrame(data) # Display the result print('DataFrame created from dictionary:') print(df)
Output:
DataFrame created from dictionary:
size bedrooms price
0 2500 3 500000
1 3000 4 600000
2 1500 2 350000
3 3500 5 750000

Turning a DataFrame Back into a Dictionary

# Convert DataFrame back to dictionary dictionary = df.to_dict('records') # Display the result print('Dictionary created from DataFrame:') print(dictionary)
Output:
Dictionary created from DataFrame:
[{'size': 2500, 'bedrooms': 3, 'price': 500000},
{'size': 3000, 'bedrooms': 4, 'price': 600000},
{'size': 1500, 'bedrooms': 2, 'price': 350000},
{'size': 3500, 'bedrooms': 5, 'price': 750000}]
Tip: The 'records' orientation in to_dict() creates a list of dictionaries, where each dictionary represents a row in the DataFrame.

Merging and Joining DataFrames

Merging and joining are operations that combine DataFrames based on common columns or indices.

Syntax

# Basic syntax for merging DataFrames pd.merge(df1, df2, on="column_name", how="join_type")

Parameters:

  • df1, df2: DataFrames to merge
  • on: Column name to join on
  • how: Type of join ('inner', 'outer', 'left', 'right')

Types of Joins

  • Inner Join: Returns only rows where key values exist in both DataFrames
  • Outer Join: Returns all rows from both DataFrames, filling missing matches with NaN
  • Left Join: Returns all rows from the left DataFrame, and matching rows from the right
  • Right Join: Returns all rows from the right DataFrame, and matching rows from the left

Example

# Creating two DataFrames import pandas as pd Customer_data = { "Name": ['Ram', 'shyam', 'lakshman', 'sheeta', 'bharat', 'shatrughan'], "customer_id": [1, 2, 3, 4, 5, 6] } df1 = pd.DataFrame(Customer_data) Order_data = { "price": [500, 700, 400, 300, 490, 910], "customer_id": [1, 2, 3, 4, 5, 8] } df2 = pd.DataFrame(Order_data) print("Customer DataFrame:") print(df1) print("\nOrder DataFrame:") print(df2) print("\n") # Inner join print("Inner Join:") merged_data = pd.merge(df1, df2, on="customer_id", how="inner") print(merged_data) print("\n") # Outer join print("Outer Join:") merged_data = pd.merge(df1, df2, on="customer_id", how="outer") print(merged_data) print("\n") # Left join print("Left Join:") merged_data = pd.merge(df1, df2, on="customer_id", how="left") print(merged_data) print("\n") # Right join print("Right Join:") merged_data = pd.merge(df1, df2, on="customer_id", how="right") print(merged_data)
Output:

Concatenating DataFrames

Concatenation is the process of joining DataFrames either vertically (row-wise) or horizontally (column-wise).

Syntax

# Basic syntax for concatenating DataFrames pd.concat([df1, df2], axis=0/1, ignore_index=True)

Parameters:

  • [df1, df2]: List of DataFrames to concatenate
  • axis=0: Concatenate vertically (row-wise)
  • axis=1: Concatenate horizontally (column-wise)
  • ignore_index=True: Reset the index of the resulting DataFrame

Row-wise Concatenation (axis=0)

# Creating two DataFrames import pandas as pd data1 = pd.DataFrame({ "Name": ['Ram', 'shyam', 'lakshman', 'sheeta', 'bharat', 'shatrughan'], "Age": [23, 45, 34, 25, 36, 29] }) data2 = pd.DataFrame({ "Name": ['Krishna', 'Arjun', 'Bheem'], "Age": [30, 35, 40] }) print("DataFrame 1:") print(data1) print("\nDataFrame 2:") print(data2) print("\n") # Row-wise concatenation result = pd.concat([data1, data2], axis=0, ignore_index=True) print("Row-wise concatenated DataFrame:") print(result)
